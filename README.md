# NextJS JWT

Aplicação web utilizando NextJS, autenticação com JWT, integrado com o banco de dados MongoDB.

## Ferramentas

* NodeJS
* React
* MongoDB

## O que contem:

* Página de login
* Página de registro
* Página de usuário

# Rotas

* / - GET
* /sign-in - GET
* /sign-up - GET

* /auth/register - POST
* /auth/login - POST
* /auth/accessToken - POST
